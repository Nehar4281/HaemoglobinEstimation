# Hemoglobin Estimation Web Application

## Description
This project estimates hemoglobin levels using machine learning.
It includes a trained ML model integrated with a web interface.

## Tech Stack
- Python
- Flask
- Machine Learning
- HTML, CSS, JavaScript

## Features
- User input-based prediction
- Trained ML model
- Simple UI

## How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
